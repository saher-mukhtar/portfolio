# Saher Mukhtar — Portfolio

This is a simple single-page portfolio built for GitHub Pages.

## Files

- `index.html` — main page
- `styles.css` — styles
- `script.js` — small interactive scripts
- `resume.pdf` — your CV (add it to repo root)

## Deploy to GitHub Pages

1. Create a repository named `portfolio`.
2. Add these files and commit.
3. Add `resume.pdf` to the repo root.
4. Go to Settings → Pages → Choose `main` branch and `/root`.
5. Your site will be available at: `https://<your-username>.github.io/portfolio`

## Customize

- Edit `index.html` to change wording, project descriptions or add images.
- Replace color values in `styles.css` to match your brand.
